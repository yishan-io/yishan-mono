**为并行思维者而生的工作区。**

Agent 让并行工作变得容易。管理它，还是靠你。Yishan 解决这个问题。

- **每个工作流都有独立的热活工作区** — 终端、Agent 会话、文件状态、git 上下文全部保持活跃。在它们之间切换，无需重新加载任何东西。
- **一个页面监控所有活跃工作** — 一眼看清所有工作流中什么变了、什么在跑、什么需要关注。
- **Worktree 级隔离** — 每个任务跑在独立的 git worktree 里，并行工作永不冲突。Yishan 让 worktree 从偶尔用的 CLI 技巧变成日常可用的工作流。
- **上下文贴在工位上** — 内置 `.my-context` 将笔记、计划、交接信息附着在每个工作区里，不卷入常规 commit。任何任务几秒就能恢复。
- **天生对 Agent 友好** — 在各自的工作区里使用你已有的 Agent 工具（OpenCode、Claude、Gemini 等）。Agent 工作与仓库和终端保持连接，不被隔离在独立聊天标签页里。

## 核心能力

- Worktree 优先的工作区管理 — 创建、监控和切换独立的分支工作区
- 多窗格开发环境 — 终端、文件编辑器、差异查看器和浏览器并排布局
- Git 变更追踪、分支可见性和 PR 状态感知
- 一键在 VS Code、Cursor、JetBrains IDE、Windsurf、Zed 或系统文件管理器中打开
- 多组织支持、成员角色和计算节点管理
- 定时 Agent 自动化和语音转录
- 运行完成/失败的应用内通知和声音提醒

## 与你已有的工具兼容

Yishan 与你已在使用的 Agent CLI 并行工作 — OpenCode、Codex、Claude、Gemini、Cursor Agent、Pi 和 Copilot。
